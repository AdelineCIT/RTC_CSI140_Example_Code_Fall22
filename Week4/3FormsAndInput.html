Forms and Input

The <form> tag is used to get input from the user and send that input to a server for processing. The action attribute defines where the data will go. The method attribute defines how the data will be sent.

A form is a collection of <label> and <input> tags. The input can either be wrapped by the label or we use an attribute on the label called for. When a label is correctly connected to an input, clicking on the label will put the cursor inside of the input.
