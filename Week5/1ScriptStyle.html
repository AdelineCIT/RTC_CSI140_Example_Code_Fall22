Script and Style

Script and Style are ways to bring JavaScript and CSS into your page.

CSS

CSS is a way to change how the elements on your page appear. There are a few ways to create CSS. The first is in line styling.

We can also style by adding a style tag to the head. This is interal styling.

We can also style from an external file that we link to in the head

JavaScript
