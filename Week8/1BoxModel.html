Box Model

Margin

Margin can be thought of as the space that surrounds the element. The margin is outsode of the border. If I want to move an element relative to its parent element. There is a good chance you should use margin

Border

Border is the space between the padding and the margin. Its invisible by default but can be set through css. It is commonly mistaken for the end of the element

Padding

Padding is the space inside of the border. It sits between then border and the content. It is the same color as the elemnt.
