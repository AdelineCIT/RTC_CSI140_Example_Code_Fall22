Tables

In html the <table> tag is used to create a table. Within a table a row is created with the <tr> tag. To create a cell in the table use either the <td> or the <th> tags.

Rowspan and Colspan

Rowspan and Colspan are attributes that can be assigned to a table cell that allow the cell to take up more space in the table.
