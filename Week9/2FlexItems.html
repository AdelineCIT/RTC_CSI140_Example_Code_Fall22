Flex Items

Individual flex items also have properties that can be changed. They can change their position, alignment or size within the flex box

  • order: The order property changes the order of the items inside the box
  • flex-grow: specifies the rate at which an element resizes itself
  • flex-shrink: Specifies how small the item will get relative to others
  • flex-basis: Setting the initial size of the element
  • flex: shortcut for flex-grow, flex-shrink and flex-basis
