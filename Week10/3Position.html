Position

The position element is used to specify how an element should be positioned relative to the elements aorund it. The location of an element can be modified once it has been given the position property. Once you set a position property You can move the element around the page using the top, bottom, left and right properties. There are 5 different values for the position property

  • Static is the default position. If an element has static I cant position
  • Relative means I can move it relative to the elements around it
  • Absolute positioning means an element has no impact on the elements around it. Any elements that are after the abolute elemtn in the document flow are going to behave as though it doesnt exist
  • Fixed position is like asbolute in that and element can go anywhere except that fixed is not affected by scrolling the document
  • Sticky is a combination of fixed and relative
  • z-index controls which element is on top when two of them are in the same spot
