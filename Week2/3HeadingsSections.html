Headings and Sections

h2 section heading

There are six heading tags <h1> through <h6> width each one getting smaller

h3 section heading

Each page should only have one <h1> tag as it shows the heading for the entire page. There can be multiples of the smaller heading tags

h4 section heading

Each of these headings are wrapped in a <section> tag

h5 section heading

the <section tag> is used to separate similar portions of webpages. These could be chapters, blog posts, products in a store etc..

h6 section heading

The <section> tag is a semantic tag. Meaning that it describes the content to the browser
