Html Lists

List in html can be created using the <ul> tag or the <ol> tag
<ul> creates an unordered list

  • Meat
  • Fruit
    • Apples
    • Bananas
    • Peaches
  • Juice

If the items need ordering use the <ol> tag

  1. Do the dishes
  2. Walk the dog
  3. Run Errands
    1. Pick up dry cleaning
    2. Buy Groceries
    3. Drop off packages
  4. Take a nap
